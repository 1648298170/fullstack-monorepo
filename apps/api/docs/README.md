## 维护约定

1. 新增关键功能时，同步补充对应使用指南。
2. 修改启动流程、环境变量、接口约定、数据库结构时，必须更新相关文档。
3. 文档示例要贴合当前项目代码，不写无法直接落地的泛用模板。
4. 文档使用中文说明，命令、路径、代码标识保持原样。
